    Details

    This Web application uses the Canvas API defined by the HTML 5 specification. You have the canvas and the tools, with their properties. You can resize the canvas using the bottom right corner handler.

    Clear image (Ctrl-N). Click the button holding the Shift key down, or press Ctrl-Shift-N, if you want to manually define the resolution of the image.

    Save image (Ctrl-S) will open a new tab/window with the image saved using the PNG format. Right-click the image and save it to your disk using the Web browser.

    Insert image does only allow data: and http:// addresses. Due to security restrictions defined in the specification, you cannot insert images from third-party hosts.

    The Text tool and the Shadow properties only work in Firefox 3.1+, since both features were recently implemented.

    Safari and Opera 9.2 (or older) do not implement the get/putImageData methods. As such, they get no support for history (Undo/Redo), no color picker, no CIE L*a*b* color visualisation, and no support for non-transparent selection backgrounds.

    Opera 9.2 does not have support for globalCompositeOperation 'lighter', which makes the RGB color visualisation to fail.
